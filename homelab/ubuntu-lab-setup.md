# Ubuntu Lab Setup

## Objective

Set up an Ubuntu virtual machine for cybersecurity labs.

## Environment

- VirtualBox
- Ubuntu ISO

## Steps

1. Created a new virtual machine.
2. Configured VM settings.
3. Installed Ubuntu.
4. Logged into the desktop.
5. Updated the system using:

```bash
sudo apt update && sudo apt upgrade -y
```
## What I Learned
- How to create Ubuntu VM.
- How to install Ubuntu manually.
- Learned how to update the system.
